# 🛍️ E-Commerce Project

A full-stack e-commerce platform developed during my **Intern Intelligence** internship. This project showcases my skills in both frontend and backend development — from building a modern, responsive storefront to designing a secure REST API and admin panel.

---

## ✨ Features

- 🛒 **Shopping cart** — add items to cart and manage quantities
- ❤️ **Favorites / Wishlist** — save products to view later
- 🔍 **Search & filters** — detailed product search and filtering options
- 📦 **Order tracking** — place orders and track their status
- 🛠️ **Admin panel** — full CRUD operations for managing products and orders
- 🔐 **Secure authentication** — JWT-based auth with hashed passwords
- ⚡ **State management** — optimized global state with Redux Toolkit
- 📱 **Responsive UI** — fast, scalable, and mobile-friendly design

---

## 🧰 Tech Stack

| Layer            | Technologies |
|------------------|--------------|
| Frontend         | Next.js 15, React 19, TypeScript, Tailwind CSS |
| State Management | Redux Toolkit, React Redux |
| UI / UX          | React Bootstrap, Headless UI, Heroicons, Lucide React, Framer Motion, Swiper |
| Notifications    | React Hot Toast, React Toastify |
| Backend          | Node.js, Express.js |
| Database         | MongoDB, Mongoose |
| Auth & Security  | JWT (jsonwebtoken), bcrypt, Helmet, CORS, express-validator |
| File Uploads     | Multer |
| Dev Tools        | Nodemon, ESLint |
